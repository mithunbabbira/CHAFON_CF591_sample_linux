RFIDTriggerReader Class Methods:
   .connect()                      - Connect to device
   .disconnect()                   - Disconnect
   .set_reading_range(meters)      - Set range (1-10 meters)
   .set_power(power)               - Set power (0-30)
   .read_once(timeout, verbose)    - Read single tag
   .read_multiple(duration)        - Read multiple tags
   .read_strongest_tag(duration)   - Find closest tag
   .set_tag_callback(callback)     - Set callback function

Tag Dictionary:
   tag['epc']        - Tag ID (hex string)
   tag['rssi']       - Signal strength (dBm)
   tag['antenna']    - Antenna number (1-4)
   tag['channel']    - Frequency channel
   tag['sequence']   - Sequence number
